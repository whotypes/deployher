# Guide

## Project layout

```
python-mkdocs-pip/
  docs/           # Markdown sources
  mkdocs.yml      # site_name, nav, theme
  requirements.txt
```

## Editing

1. Add a `.md` file under `docs/`.
2. Register it under `nav` in `mkdocs.yml`.
3. Rebuild — MkDocs emits a static `site/` (or your configured output) for Deployher.
